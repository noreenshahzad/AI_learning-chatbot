🤖 AI Learning Chatbot

This project is an AI-based learning chatbot that scrapes Artificial Intelligence and Machine Learning related information from Wikipedia and provides a simple interactive chatbot interface.

📌 Project Description

This chatbot collects data from Wikipedia pages related to Artificial Intelligence and Machine Learning, processes the text, and allows users to ask questions based on the scraped knowledge.

🛠️ Technologies Used
Python
BeautifulSoup
Requests
Streamlit
⚙️ How to Run
1️⃣ Install Requirements
pip install -r requirements.txt
2️⃣ Run Web Scraper
python web_scraper.py
3️⃣ Run Chatbot (Streamlit App)
streamlit run app.py
🎯 Features
Web scraping from Wikipedia
AI & ML knowledge base creation
Interactive chatbot interface
Simple and user-friendly UI
👩‍💻 Purpose of Project

To learn and demonstrate the basics of:

Web Scraping
Natural Language Processing (NLP)
AI-based chatbot development
